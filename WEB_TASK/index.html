HTML 은 무엇인가요?

HTML은 HyperText Markup Language의 약자로, 웹 페이지의 구조와 내용을 정의하는 데 사용되는 마크업 언어입니다. 텍스트, 이미지, 동영상 등 다양한 콘텐츠를 웹 페이지에 표시하고 사용자와의 상호 작용을 가능하게 하는 역할을 합니다.

CSS 는 무엇인가요?

CSS(Cascading Style Sheets)는 HTML로 작성된 문서의 스타일을 정의하는 데 사용되는 스타일 시트 언어입니다. 텍스트 크기, 색상, 글꼴, 여백, 배치 등 다양한 시각적 요소를 제어하여 웹 페이지의 디자인과 레이아웃을 지정하는 데 중요한 역할을 합니다.
