1. CSS 란 무엇인가?

CSS는 Cascading Style Sheets의 약자로서 HTML로 작성된 문서의 시각적 스타일을 정의하는 데 사용되는 스타일 시트 언어입니다.

HTML은 웹 페이지의 구조와 내용을 정의하는 반면, CSS는 페이지의 글꼴, 색상, 레이아웃, 여백, 폰트 크기 등을 지정하여 페이지를 보다 아름답고 보기 쉽게 만들어 줍니다.

CSS를 사용하면 웹 페이지의 디자인을 일관되게 유지하고, 다양한 디바이스(예: 컴퓨터, 스마트폰, 태블릿)에 맞게 페이지를 최적화할 수 있습니다. 또한, JavaScript와 함께 사용하여 웹 페이지에 동적인 기능을 추가할 수도 있습니다.

2. CSS의 주요 특징

- HTML과 분리된 스타일 정의: CSS는 HTML 코드와 별도로 작성되므로, HTML 코드를 변경하지 않고도 페이지의 스타일을 변경할 수 있습니다.

- 다양한 스타일 속성: CSS는 글꼴, 색상, 여백, 패딩, 테두리, 위치, 디스플레이 등 다양한 스타일 속성을 제공합니다.

- 선택자를 통한 정확한 스타일 적용: CSS는 특정 요소, 그룹 또는 페이지의 모든 요소를 선택하여 스타일을 적용할 수 있습니다.

- 계층적 스타일 시스템: CSS는 계층적 스타일 시스템을 사용하여 스타일 충돌을 해결하고, 더욱 효율적으로 스타일을 적용합니다.

- 다양한 웹 브라우저 지원: CSS는 대부분의 웹 브라우저에서 지원됩니다.

3. CSS 요소 예시

CSS는 웹 페이지의 디자인을 정의하는 데 사용되는 스타일 시트 언어입니다. 다양한 요소들을 사용하여 웹 페이지의 글꼴, 색상, 레이아웃 등을 조정할 수 있습니다.

(1) 선택자 (Selector)

어떤 요소에 스타일을 적용할지 결정합니다.

CSS 규칙의 첫 번째 부분이며, 클래스, ID, 태그 이름, 속성 등을 사용하여 요소를 선택할 수 있습니다.

예시: h1, .container, #main

(2) 속성 (Property)

요소의 특정 디자인 요소를 정의합니다.

글꼴, 색상, 크기, 여백, 패딩, 테두리 등 다양한 속성이 존재합니다.

예시: font-family, color, font-size, margin, padding, border

(3) 값 (Value)

속성의 특정 값을 지정합니다.

숫자, 색상 코드, 길이 단위, 백분율 등 다양한 형식의 값을 사용할 수 있습니다.

예시: Arial, #FF0000, 12px, 10px, 2px solid #ccc

(4) 규칙 (Rule)

선택자, 속성, 값을 조합하여 하나의 CSS 규칙을 구성합니다.

; 기호로 구분되며, 여러 규칙을 {} 안에 작성하여 하나의 블록으로 묶을 수 있습니다.

(5) 선언 블록 (Declaration Block)

{} 안에 작성되는 CSS 규칙들의 집합입니다.

같은 선택자에 속하는 여러 규칙을 한 번에 정의할 수 있습니다.
